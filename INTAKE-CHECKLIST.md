## Processing Workflow

### Step 1: Gather
- [ ] Locate all materials (local files, cloud storage, email attachments)
- [ ] Download files from Google Drive/Dropbox/OneDrive
- [ ] Export Google Docs/Sheets as needed
- [ ] Save email attachments
- [ ] Request assets from David & Kirsty (logos, covers, photos)

### Step 2: Review
- [ ] Check file quality (resolution, format, completeness)
- [ ] Identify duplicates
- [ ] Note missing materials
- [ ] Assess what needs creation vs. exists

### Step 3: Rename
Use consistent naming:
- Lowercase letters
- Hyphens instead of spaces
- Include date if relevant: `filename-YYYYMMDD.ext`
- Version numbers: `filename-v1.0.ext`

Examples:
- `book-cover-grey-coast-2026.jpg`
- `author-photo-beth-mcdonough.jpg`
- `logo-north-house-press.svg`
- `meeting-transcript-20251023.md`

### Step 4: Organize
- [ ] Move files to appropriate directories
- [ ] Create subdirectories as needed
- [ ] Update `assets/asset-inventory.md`
- [ ] Add links to `resource-links.md` if cloud-hosted

### Step 5: Document
- [ ] Update asset inventory
- [ ] Add notes about file sources
- [ ] Document any access restrictions
- [ ] Note if files need further processing

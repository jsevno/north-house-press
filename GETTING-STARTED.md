## üí° Tips & Best Practices

### File Management
- ‚úÖ Use descriptive names
- ‚úÖ Include dates when relevant (YYYYMMDD format)
- ‚úÖ Keep versions organized (v1.0, v1.1, etc.)
- ‚ùå Don't use spaces in filenames
- ‚ùå Don't duplicate files across folders

### Documentation
- Write in UK English
- Use Markdown format (.md) for text documents
- Keep things factual and evidence-based
- Cross-reference related documents
- Update "Last Updated" dates

### Website Development
- **Interim site:** Quick iterations, show progress
- **WordPress site:** Plan carefully, build once
- Keep both sites in sync with project status
- Use interim site to test designs before WordPress
